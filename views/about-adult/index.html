Obtemos o Adult Data Set do UCI Machine Learning Repository, que é uma coleção de bancos de dados, teorias de domínio e geradores de dados que são usados pela comunidade de machine learning para a análise empírica de algoritmos de machine learning. Para saber mais, acesse UCI Machine Learning Repository.

Em resumo, o Adult Data Set busca prever se a renda de adultos (aqueles com maior de 16 anos) de 1994 excede US$ 50 mil por ano com base nos dados do censo, também conhecido como conjunto de dados "Census Income". A extração dos dados foi feita por Barry Becker. Para obter mais detalhes, acesse Adult Data Set.
